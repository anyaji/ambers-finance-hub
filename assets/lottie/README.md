# Drop your Lottie animations here 🎬

The app **auto-detects** any of these files in this folder. No code changes needed —
just save the file with the exact name and refresh the app.

| File name | Where it shows |
|-----------|----------------|
| `mascot.json` | The little character on the **Home** hero (replaces the emoji 🐷) |
| `celebrate.json` | A **full-screen overlay** that plays on level-ups, badges & milestones |
| `house.json` | Reserved for a future goal animation |

If a file isn't here, the app simply uses the built-in emoji + confetti effects.

## Where to get `.json` animations (free)
- **lottiefiles.com** — search "piggy bank", "new home", "celebration", "confetti".
  Click an animation → **Download → Lottie JSON** → rename to one of the names above.
- Or export Lottie JSON from **Glaxnimate**, **Rive**, **Cavalry**, or **After Effects**
  (Bodymovin plugin).

## Tips
- Keep files reasonably small (under ~500 KB) so the app stays snappy.
- `mascot.json` looks best as a looping square animation.
- `celebrate.json` should be a short one-shot (it plays once, then disappears).
- Test from the app: **Settings → Custom Animations → 🎉 Test celebration**.
